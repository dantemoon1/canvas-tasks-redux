    About

    Tasks for Canvas Redux is a community fork of Tasks for Canvas by Jeffrey Cheng, rebuilt with a modern permissions model and zero tracking.

    • You control which Canvas domains are enabled
    • No analytics or remote data collection
    • Original task sidebar features preserved
